# JudolGraph Demo Web

Vite + React + TypeScript demo app for JudolGraph, an AI-powered investigation graph platform for documenting synthetic online-gambling investigation evidence.

All demo data is synthetic and intended for UI presentation only.

## Run Locally

```bash
pnpm install
pnpm dev
```

Open:

```txt
http://localhost:5173/
```

## Useful Scripts

```bash
pnpm lint
pnpm build
pnpm preview
```

## Demo Flow

1. Landing page
2. Dashboard
3. Evidence Graph
4. Entity Detail
5. Screenshot & OCR Evidence
6. Report Export
7. Mobile Case Overview
